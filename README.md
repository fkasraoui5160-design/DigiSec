# DigiSec

**DigiSec** est une application web offrant plusieurs services de sécurité simples mais fonctionnels, développée pour démontrer les bases de la cybersécurité appliquée.

## Services proposés
1. **Signature manuelle (stamp)** : permet de signer manuellement des documents ou données.  
2. **Chiffrement** : chiffrement des fichiers pour garantir la confidentialité.  
3. **Certification et signature numérique** : génération et vérification de certificats numériques.  
4. **Horodatage (timestamp)** : service d’horodatage sécurisé pour documents et fichiers.  
5. **Envoi de mails chiffrés** : envoi de courriels avec chiffrement pour protéger les informations.  
6. **Création de QR codes chiffrés** : génération de QR codes contenant des données sécurisées.

## Technologies utilisées
- Node.js
- HTML / CSS
- HTTPS pour la sécurité des échanges

## Structure du projet
- `uploads/` : fichiers téléchargés
- `stamp/` : données liées à la signature manuelle
- `testimonial.html` : page de démonstration
- `package.json` : configuration Node.js (si applicable)
